## Pages/Features

- First-time onboarding flow
- Dashboard with progress, next steps, deadlines, and alerts
- Academic roadmap / My Plan section
- Course planning and course details screens
- Alerts, errors, and prerequisite recovery states
- Resources and support screen
- Confirmation and feedback screens
- Responsive and accessible interaction patterns

## Setup

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## Deployment

Target platform: Vercel
